NUMEROS INTEIROS ( Z )


O conjunto dos numeros inteiros e representado por

Z = {... -4, -3, -2, -1, 0, 1, 2, 3, 4, ...}

o conjunto dos naturais e subconjunto dos inteiros!
Logo, os naturais estao "dentro" dos inteiros.

ℕ ⊂ ℤ

Temos ainda o conjunto dos inteiros nao negativos
que e representado por Z+
em que inclui todos os inteiros positivos mais o zero.
E temos os inteiros nao positivos,
que inclui todos os negativos mais o zero.

Z+ = { 0, 1, 2, 3, 4, 5, 6, ...} Inteiros NAO negativos

Z- = { ... -6, -5, -4, -3, -2, -1, 0 } Inteiros NAO negativos

Se excluirmos o zero desses conjuntos anteriores,
teremos os inteiros positivos e inteiros negativos:

Z*+ = { 1, 2, 3, 4, 5, 6, ...} Inteiros NAO negativos sem o zero

Z*- = { ... -6, -5, -4, -3, -2, -1 } Inteiros negativos sem o zero

Todos esses conjuntos sao subconjuntos de Z!

Simetrico e Oposto

O oposto ou simetrico de +5 e -5.
Isso significa que tanto o +5 como o -5 estao a mesma distancia do zero.
adicao de um numero inteiro ao seu oposto,
resulta em zero: (+3) + (-3) = + 3 – 3 = 0



Regras de sinais:
Adicao e subtracao entre dois numeros:
✓ sinais iguais => soma se e mantem o sinal.
✓ sinais diferentes => subtrai se e mantem o sinal do numero de maior modulo.
Multiplicacao e divisao entre dois numeros:
✓ sinais iguais => o sinal fica positivo.
✓ sinais diferentes => o sinal fica negativo.

Ha operacoes de divisao em Z que resultam em valores que nao fazem parte de Z
Por exemplo: a divisao de -3 por 2, sabemos que resulta em -1,5.
